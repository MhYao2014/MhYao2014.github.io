з¬¬е…ӯз«  ж”ҜжҢҒеҗ‘йҮҸжңә

6.1 ж”ҜжҢҒеҗ‘йҮҸжңәзҡ„еҹәжң¬еһӢ

Tips 1пјҡзӮ№еҲ°и¶…е№ійқўзҡ„и·қзҰ»е…¬ејҸ

з»ҷе®ҡи¶…е№ійқў $W^{\top}x+b=0$ еҗҺпјҢи®Ўз®—жҹҗдёҖзӮ№ $A=x_{i}$ еҲ°иҜҘе№ійқўзҡ„и·қзҰ»жҺЁеҜјпјҡ

и®ҫи·қзҰ»дёә$r$ , еҲҷеҸҜд»ҘйҖҡиҝҮдёӢйқўзҡ„зӯүејҸи§ЈеҮә$r$ :

иҝҷжҳҜеӣ дёә$A$еңЁи¶…е№ійқўдёҠзҡ„жҠ•еҪұзӮ№$B$еҸҜд»ҘиЎЁзӨәдёәпјҡ

иҝҷйҮҢзҡ„еҮ дҪ•ж„Ҹд№үеҫҲжҳҺжҳҫгҖӮдёҚеҶҚиөҳиҝ°гҖӮ

жүҖд»Ҙж•ҙзҗҶж–№зЁӢ$\eqref{pluginpoint}$еҸҜд»Ҙеҫ—еҲ°пјҡ

Tips 2пјҡй—ҙйҡ”зҡ„жҰӮеҝө

еҒҮи®ҫжүҖжңүзӮ№йғҪе·Із»ҸиғҪиў«и¶…е№ійқў$l:W^\top x + b = 0$еҲҶзұ»жӯЈзЎ®пјҢйӮЈд№ҲеҜ№жүҖжңүзӮ№$x_i$йғҪжңүеҜ№еә”зҡ„$\epsilon_i>0$пјҢдҪҝеҫ—дёӢејҸжҲҗз«Ӣпјҡ

йҖҡиҝҮеңЁдёҠиҝ°дёҚзӯүејҸдёӯпјҢйҰ–е…ҲеҜ№$W$е’Ң$b$ж–ҪеҠ йҖӮеҪ“зҡ„ж—ӢиҪ¬еҸҳжҚўпјҢиҖҢеҗҺеҗҢж—¶йҷӨд»Ҙ $\min\limits_{i}\epsilon_i$ е°ұжҖ»иғҪдҪҝеҫ—дёӢејҸжҲҗз«Ӣпјҡ

дёӢйқўзқҖйҮҚиҖғеҜҹйӮЈдәӣдҪҝеҫ—дёҠиҝ°дёҚзӯүејҸеҸ–еҫ—зӯүеҸ·зҡ„ж•°жҚ®зӮ№пјҢеҒҮи®ҫ$x{i{n}}^+$еңЁи¶…е№ійқў$l^+:W^\top x + b = +1$дёҠпјӣ$x{i{m}}^-$еңЁи¶…е№ійқў$l^-:W^\top x + b = -1$дёҠгҖӮйӮЈд№Ҳжҳҫз„¶иҝҷдёӨдёӘзӮ№жҳҜжүҖжңүж•°жҚ®зӮ№дёӯйҮҢеҲҶзҰ»и¶…е№ійқў$W^\top x + b = 0$жңҖиҝ‘зҡ„дёӨдёӘзӮ№гҖӮйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»Ҙе®ҡд№үвҖңй—ҙйҡ”вҖқиҝҷж ·дёҖдёӘжҰӮеҝөпјҡ$l^+$дёҺ$l^-$д№Ӣй—ҙзҡ„и·қзҰ»з§°дёәвҖңй—ҙйҡ”вҖқпјҲmarginпјүпјҢиҝҷйҮҢжҲ‘д»¬и®°дҪң$\gamma$гҖӮжҳҫз„¶$l^+$дёҺ$l^-$д№Ӣй—ҙзҡ„и·қзҰ»дёә$l^+$еҲ°$l$зҡ„и·қзҰ»еҠ дёҠ$l$еҲ°$l^-$зҡ„и·қзҰ»пјҲд№ҹе°ұжҳҜ$x{i{n}}^+$еҲ°$l$зҡ„и·қзҰ»еҠ дёҠ$x{i{m}}^-$еҲ°$l$зҡ„и·қзҰ»пјүгҖӮ

ж №жҚ®и·қзҰ»е…¬ејҸ$\eqref{distance}$пјҢ$x{i{n}}^+$еҲ°$l$зҡ„и·қзҰ»дёәпјҡ

дёҠејҸз¬¬дәҢдёӘзӯүеҸ·жҳҜеӣ дёә$x{i{n}}^+$иҗҪеңЁдәҶ$l^+$дёҠпјҢжүҖд»ҘеҸҜд»Ҙе°Ҷ$|W^\top x{i{n}}^++b|=1$д»Је…ҘдёҠејҸгҖӮ

еҗҢзҗҶжҲ‘д»¬жңү

жүҖд»Ҙй—ҙйҡ”$\gamma=\frac{2}{|W|}$гҖӮ

Tips 3: ж”ҜжҢҒеҗ‘йҮҸжңәеҹәжң¬еһӢзҡ„еҜјеҮә

ж”ҜжҢҒеҗ‘йҮҸжңәе»әжЁЎзҡ„жғіжі•еңЁдәҺиҰҒжңҖеӨ§еҢ–жүҫеҲ°зҡ„йӮЈдёӘи¶…е№ійқўжүҖеҜ№еә”зҡ„вҖңй—ҙйҡ”вҖқгҖӮжҳҫз„¶пјҢиҝҷж ·жүҫеҲ°зҡ„и¶…е№ійқўеңЁе°Ҹж ·жң¬дёӢдјҡжңүжӣҙеҠ йІҒжЈ’зҡ„иЎЁзҺ°гҖӮйӮЈд№ҲеҺҹе§Ӣзҡ„дјҳеҢ–й—®йўҳе°ұдјҡеҶҷжҲҗдёӢйқўзҡ„еҪўејҸпјҡ

иҖғиҷ‘еҲ°дёҠйқўзҡ„дјҳеҢ–й—®йўҳе№¶дёҚжҳҜеҮёеҮҪж•°пјҢдёҚиғҪж–№дҫҝзҡ„жұӮи§ЈпјҢиҖғиҷ‘е°ҶжңҖеӨ§еҢ–$\frac{2}{|W|}$ж”№дёәжңҖе°ҸеҢ–$\frac{1}{2}|W|^2$гҖӮиҝҷдёӨдёӘдјҳеҢ–й—®йўҳеңЁж•°еӯҰдёҠзӯүд»·пјҢд№ҹе°ұжҳҜеҗҢи§ЈгҖӮжүҖд»ҘжҲ‘д»¬еҜјеҮәдәҶж”ҜжҢҒеҗ‘йҮҸжңәзҡ„еҹәжң¬еһӢпјҢд№ҹз§°д№ӢдёәеҺҹй—®йўҳпјҡ

е…¶е®һеңЁиҝҷйҮҢе·Із»ҸеҸҜд»ҘйҖҡиҝҮд»Јз ҒеҢ…жқҘи§ЈиҝҷдёӘдјҳеҢ–й—®йўҳдәҶпјҒеҸӘжҳҜи§Јиө·жқҘпјҢж•ҲзҺҮдјҡдҪҺдёҖдәӣгҖӮ

6.2 ж”ҜжҢҒеҗ‘йҮҸжңәеҺҹй—®йўҳзҡ„еҜ№еҒ¶й—®йўҳ

Tips 1: дёәд»Җд№Ҳз ”з©¶еҜ№еҒ¶й—®йўҳе’Ң KKT жқЎд»¶

еҜ№дәҺеҲқж¬ЎжҺҘи§ҰдјҳеҢ–й—®йўҳзҡ„еҗҢеӯҰиҖҢиЁҖпјҢеҜ№еҒ¶й—®йўҳдёҖиҲ¬жҳҫеҫ—иҺ«еҗҚеҘҮеҰҷгҖӮжңҖзӣҙжҺҘзҡ„й—®йўҳжҳҜпјҢжҲ‘д»¬дёәд»Җд№ҲиҰҒз ”з©¶еҜ№еҒ¶й—®йўҳпјҹиҝӣдёҖжӯҘең°пјҢжңүзҡ„дәәеҸҜиғҪиҝҳзҹҘйҒ“жҲ‘д»¬иҝҳиҰҒжҗһдёҖдёӘKKTжқЎд»¶пјҢз ”з©¶йӮЈдёӘеҸҲе№Іеҳӣпјҹ

иҝҷеҮ дёӘй—®йўҳеңЁж”ҜжҢҒеҗ‘йҮҸжңәзҡ„иғҢжҷҜдёӢиҝҳжҳҜеҫҲеҘҪеӣһзӯ”зҡ„пјҢйҖ»иҫ‘жҳҜиҝҷж ·зҡ„пјҡеңЁж”ҜжҢҒеҗ‘йҮҸжңәзҡ„жЎҲдҫӢдёӢпјҢеҜ№еҒ¶й—®йўҳзҡ„и§ЈиҝҳжҳҜеҺҹй—®йўҳзҡ„и§ЈпјҢе№¶дё”еҜ№еҒ¶й—®йўҳжҜ”еҺҹй—®йўҳ$\eqref{primal}$иҰҒеҘҪи§ЈеҶіпјӣдёәд»Җд№ҲеҜ№еҒ¶й—®йўҳзҡ„и§ЈдјҡжҳҜеҺҹй—®йўҳзҡ„и§Је‘ўпјҹеӣ дёәеҜ№еҒ¶й—®йўҳзҡ„и§Јж»Ўи¶іеҺҹй—®йўҳзҡ„KKTжқЎд»¶пјӣдёәд»Җд№ҲеҜ№еҒ¶й—®йўҳеҘҪи§ЈеҶіе‘ўпјҹеӣ дёәеңЁз ”з©¶еҜ№еҒ¶й—®йўҳиҝҮзЁӢдёӯеҸҜд»ҘйҖҡиҝҮеҺҹй—®йўҳзҡ„KKTжқЎд»¶еҺ»йҷӨжҺүеӨ§еӨҡж•°зҡ„ж•°жҚ®зӮ№пјҢжүҫеҲ°еҜ№жңҖдјҳи§Јиө·дҪңз”Ёзҡ„йӮЈеҮ дёӘж•°жҚ®зӮ№пјҢе№¶еҸӘеҲ©з”Ёе®ғд»¬иҝӣиЎҢи®Ўз®—пјҢд»ҺиҖҢйҷҚдҪҺдәҶй—®йўҳзҡ„йҡҫеәҰгҖӮиҝҷдёӘйҖ»иҫ‘еӨ§е®¶е…ҲжҺҘж”¶дёӢпјҢдёӢйқўжҲ‘дёҖжӯҘдёҖжӯҘзҡ„showеҮәжқҘгҖӮ

йҰ–е…ҲжқҘдёҖеӨ§дёІе®ҡд№үпјҢеӨ§е®¶иғҢдҪҸе°ұеҘҪпјҒ

й«ҳж•°дёӯеӯҰиҝҮеёҰзӯүејҸзәҰжқҹзҡ„дјҳеҢ–й—®йўҳпјҢеҸҜд»ҘеҲ©з”ЁжӢүж јжң—ж—Ҙж•°д№ҳжі•ж¶ҲеҺ»зӯүејҸзәҰжқҹгҖӮеҗҢж ·зҡ„пјҢдёҚзӯүејҸзәҰжқҹд№ҹеҸҜд»ҘеҲ©з”Ёзұ»дјјзҡ„жҠҖе·§ж¶ҲеҺ»гҖӮеңЁж¶ҲеҺ»зҡ„иҝҮзЁӢдёӯпјҢжҲ‘д»¬дјҡеҫ—еҲ°жӢүж јжң—ж—ҘеҮҪж•°гҖӮиҝҷдёӘжӢүж јжң—ж—ҘеҮҪж•°жҳҜжҲ‘д»¬жһ„йҖ еҜ№еҒ¶й—®йўҳзҡ„еҹәзЎҖпјҒеӨ§е®¶з¬¬дёҖжӯҘиө·жүӢејҸиҰҒз»ғжүҺе®һпјҢйғҪжҳҜйқ иғҢзҡ„еҠҹеӨ«пјҒ

Tips 2: еҰӮдҪ•еҜјеҮәеҜ№еҒ¶й—®йўҳ

йҰ–е…Ҳй’ҲеҜ№еҺҹй—®йўҳ$\eqref{primal}$пјҢжҲ‘д»¬е®ҡд№үе®ғзҡ„жӢүж јжң—ж—ҘеҮҪж•°пјҡ

е…¶дёӯиҰҒжұӮ$\alpha_i\ge0$гҖӮ

жҺҘзқҖеҜ№еӣәе®ҡзҡ„$\alpha_i$пјҢйҖҡиҝҮеҜ№$W,b$жұӮжӢүж јжң—ж—ҘеҮҪж•°зҡ„жңҖе°ҸеҖјпјҢжҲ‘д»¬е®ҡд№үеҜ№еҒ¶еҮҪж•°пјҡ

дёҠиҝ°жұӮжңҖе°ҸеҖјзҡ„иҝҮзЁӢиҰҒжұӮжҲ‘д»¬еҜ№$W,b$еҗҢж—¶жұӮеҒҸеҜјпјҡ

и§Јеҫ—$W=\sum_{i=1}^{m}\alpha_iy_ix_i$гҖӮе°Ҷиҝҷдәӣз»“жһңеёҰеӣһеҲ°$\eqref{largerange}$ еҸҜд»Ҙеҫ—еҲ°иҜҰз»Ҷзҡ„еҜ№еҒ¶еҮҪж•°пјҡ

еҜ№еҒ¶й—®йўҳе°ұжҳҜжңҖеӨ§еҢ–еҜ№еҒ¶еҮҪж•°пјҡ

дёҠйқўзҡ„еҜ№еҒ¶й—®йўҳи§ЈеҮә$\alpha_{i}$еҗҺеҶҚеёҰеӣһеҲ°$\eqref{min}$пјҢе°ұеҸҜд»Ҙеҫ—еҲ°$W,b$гҖӮиҝҷйҮҢжңүдёҖдёӘй—®йўҳпјҢдёәд»Җд№Ҳиҝҷж ·еҫ—еҲ°зҡ„$W,b$е°ұжҳҜеҺҹй—®йўҳзҡ„жңҖдјҳи§ЈпјҹдёӢйқўзҡ„KKTжқЎд»¶дјҡShowеҮәиҝҷдёҖзӮ№пјҒ

Tips 3: KKTжқЎд»¶зҡ„дҪңз”Ё

дёӢйқўжқҘshowдёҖдёӢKKTжқЎд»¶еҲ°еә•жңүд»Җд№Ҳз”ЁгҖӮ

йҰ–е…ҲжҳҺзЎ®дёҖзӮ№пјҢдёҠиҝ°еҜ№еҒ¶й—®йўҳе’ҢKKTжқЎд»¶жІЎжңүеҝ…з„¶иҒ”зі»гҖӮжҲ‘д»¬дёҚж¶үеҸҠеҜ№еҒ¶й—®йўҳд№ҹеҸҜд»Ҙи®Ёи®әеҺҹй—®йўҳзҡ„KKTжқЎд»¶гҖӮиҝҷдёҖдёӘTipsе°ұжҳҜиҰҒshowдёҖдёӢеҺҹй—®йўҳзҡ„KKTжқЎд»¶жҳҜеҰӮдҪ•её®еҠ©еӣһзӯ”дёҠйқўдёӨдёӘй—®йўҳзҡ„пјҡдёәд»Җд№Ҳж”ҜжҢҒеҗ‘йҮҸжңәдёӯеҜ№еҒ¶й—®йўҳдёҺеҺҹй—®йўҳеҗҢи§Јпјӣдёәд»Җд№ҲеҜ№еҒ¶й—®йўҳжӣҙе®№жҳ“и§ЈеҶігҖӮ

е…¶ж¬ЎпјҢдёӢйқўеҜ№еҺҹй—®йўҳзҡ„KKTжқЎд»¶еұ•ејҖи®Ёи®әпјҡ

еңЁжҲ‘д»¬жұӮи§ЈеҺҹй—®йўҳ$\eqref{primal}$ж—¶пјҢдјҡйҖҡиҝҮжӢүж јжң—ж—Ҙд№ҳеӯҗжі•жһ„йҖ еҮәжӢүж јжң—ж—ҘеҮҪж•°$L(\alpha_i, W, b)$пјҢе№¶дё”и§ЈдёӢйқўзҡ„дјҳеҢ–й—®йўҳпјҲеңЁеҮёеҮҪж•°зҡ„иғҢжҷҜдёӢвҖ”$\frac{1}{2}|W|^2$жҳҫз„¶жҳҜеҮёеҮҪж•°вҖ”иҝҷдёӘдјҳеҢ–й—®йўҳзҡ„и§Је’ҢеҺҹй—®йўҳеҗҢи§Јпјүпјҡ

иҝҷдёӘй—®йўҳе’ҢеҜ№еҒ¶й—®йўҳй•ҝзҡ„еҫҲеғҸпјҢдҪҶзЎ®е®һдёҚжҳҜеҜ№еҒ¶й—®йўҳ23333гҖӮжӯӨж—¶ж №жҚ®KKTжқЎд»¶зҡ„е®ҡд№үпјҢжҲ‘д»¬е°ұеҸҜд»ҘеҜ№дёҠйқўзҡ„дјҳеҢ–й—®йўҳ$\eqref{largerangianpro}$еҲ—еҮәе®ғзҡ„KKTжқЎд»¶пјҲжңҖдјҳи§Јеҝ…йЎ»ж»Ўи¶ізҡ„жқЎд»¶пјүпјҢеҗҢж—¶еӣ дёәеҗҢи§Јзҡ„зјҳж•…пјҢеҺҹй—®йўҳзҡ„и§Јд№ҹеҝ…з„¶ж»Ўи¶іпјҡ

йҖҡиҝҮи§ӮеҜҹеҸҜд»ҘеҸ‘зҺ°дёҠиҝ°з¬¬дёҖжқЎе’Ңз¬¬дәҢжқЎзӯүејҸе°ұжҳҜеҜ№еҒ¶й—®йўҳжһ„йҖ иҝҮзЁӢдёӯиҰҒжұӮж»Ўи¶ізҡ„жқЎд»¶$\eqref{min}$пјӣз¬¬3жқЎдёҺз¬¬4жқЎдёәд»Җд№Ҳж»Ўи¶іиҝҳжІЎи§ӮеҜҹеҮәжқҘгҖӮ
